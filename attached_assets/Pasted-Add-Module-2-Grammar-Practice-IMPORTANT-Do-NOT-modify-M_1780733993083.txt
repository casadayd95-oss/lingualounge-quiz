MODULE 2 GRAMMAR TOPICS

Use only grammar and structures from Module 2 (Chapters 4–6).

1. Verb Conjugation — sehen

Examples:

Ich _____ den Tisch.
→ sehe

Du _____ das Bild.
→ siehst

Er _____ die Lampe.
→ sieht

Wir _____ den Kalender.
→ sehen

2. Definite Articles — Nominativ

Practice:
der / die / das / die plural

Examples:

Das ist _____ Tisch.
→ der

Das ist _____ Lampe.
→ die

Das ist _____ Bild.
→ das

Das sind _____ Bücher.
→ die

3. Indefinite Articles — Nominativ

Practice:
ein / eine

Examples:

Das ist _____ Tisch.
→ ein

Das ist _____ Uhr.
→ eine

Das ist _____ Buch.
→ ein

4. Negative Articles — Nominativ

Practice:
kein / keine / kein / keine plural

Examples:

Das ist _____ Tisch.
→ kein

Das ist _____ Lampe.
→ keine

Das ist _____ Bild.
→ kein

Das sind _____ Termine.
→ keine

5. Definite Articles — Akkusativ

Practice:
der → den

Examples:

Ich sehe _____ Tisch.
→ den

Ich sehe _____ Lampe.
→ die

Ich sehe _____ Bild.
→ das

Ich sehe _____ Bücher.
→ die

6. Indefinite Articles — Akkusativ

Practice:
einen / eine / ein

Examples:

Ich habe _____ Kalender.
→ einen

Ich brauche _____ Tasche.
→ eine

Ich sehe _____ Tablet.
→ ein

7. Negative Articles — Akkusativ

Practice:
keinen / keine / kein

Examples:

Ich habe _____ Termin.
→ keinen

Ich brauche _____ Tasche.
→ keine

Ich sehe _____ Tablet.
→ kein

8. Singular → Plural

Use Module 2 vocabulary only.

Examples:

das Buch → die Bücher

der Laptop → die Laptops

die Maus → die Mäuse

das Passwort → die Passwörter

die Nachricht → die Nachrichten

das Tablet → die Tablets

9. Personal Pronouns

Practice:
er / sie / es / sie plural

Examples:

der Stuhl → er

die Lampe → sie

das Bild → es

die Bücher → sie

10. Preposition aus

Examples:

Die Flasche ist _____ Plastik.
→ aus

Der Tisch ist _____ Holz.
→ aus

Die Brille ist _____ Metall.
→ aus

11. Asking Prices

Examples:

_____ kostet der Stuhl?
→ Wie viel

Was kostet das Sofa?
→ Es kostet 399 Euro.

12. Describing Objects

Examples:

Der Stuhl ist nicht schön.
→ hässlich

Das Sofa ist nicht teuer.
→ günstig

Der Schrank ist nicht alt.
→ modern

Das Regal ist groß und _____.
→ praktisch

13. Verb Usage — finden

Examples:

Ich _____ das Sofa schön.
→ finde

Wir _____ den Preis gut.
→ finden

14. Verb Usage — kosten

Examples:

Das Buch _____ zehn Euro.
→ kostet

Der Schrank _____ 99 Euro.
→ kostet
